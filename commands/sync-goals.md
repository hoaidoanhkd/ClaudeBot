# Sync Goals with GitHub Issues

Synchronize GOALS.md with GitHub Issues for the configured project.

## Usage:

### Push (GOALS.md -> GitHub Issues)
```bash
~/scripts/goals-sync.sh --push
```

### Pull (GitHub Issues -> GOALS.md)
```bash
~/scripts/goals-sync.sh --pull
```

### Both directions
```bash
~/scripts/goals-sync.sh --both
```

Report: how many issues created, how many goals updated.
