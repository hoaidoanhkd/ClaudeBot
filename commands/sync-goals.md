# Sync Goals ↔ GitHub Issues

Đồng bộ GOALS.md với GitHub Issues.

## Thực hiện:

### Push (GOALS.md → GitHub Issues)
```bash
~/scripts/goals-sync.sh --push
```

### Pull (GitHub Issues → GOALS.md)
```bash
~/scripts/goals-sync.sh --pull
```

### Both
```bash
~/scripts/goals-sync.sh --both
```

Sau đó báo cáo: bao nhiêu issues tạo mới, bao nhiêu goals cập nhật.
